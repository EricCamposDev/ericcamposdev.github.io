Atomo Framework

Que tal construir um projeto de site ou sistema usando o conhecimento basico de orientação a objeto? sim, isso é possivel, e o que a unica coisa que te limita é um simples CLONE de GIT ou download do ADK(Atomo Development Kit).

  Baixar ADK  Documentação

Instalação linux:

1 - entre na pasta do servidor

~ $ cd /var/www
ou
~ $ cd /var/www/html

2 - de um clone do projeto

~ $ git clone https://github.com/EricCamposDev/atomo.git

3 - abra a pasta /atomo do seu servidor web em algum navegador de sua preferência

~ ex: http://localhost/atomo

Servidor Linux

Desenvolvido e testado de forma nativa em servidores linux.

Servidor Windows

Ainda não recomendado, pois algums componentes ainda estão sendo adaptados para este sistema operacional.

Servidor Mac
